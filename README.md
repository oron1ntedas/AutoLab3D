# AutoLab3D: Десктопная среда для создания виртуальных лабораторных работ


**AutoLab3D** — это десктопное приложение для проектирования и проведения виртуальных экспериментов. Приложение работает как «Тонкий клиент»: основные вычисления и хранение библиотеки моделей происходят в облаке, а рендеринг и взаимодействие — на локальной машине пользователя для максимальной производительности.

---

## Архитектура решения

Проект построен по клиент-серверной архитектуре:

1.  **Desktop Client (Unity):**
    * Отвечает за рендеринг высокополигональных 3D-сцен.
    * Обеспечивает физику взаимодействия (PhysX).
    * Кэширует загруженные лабораторные работы для оффлайн-доступа.
2.  **Cloud Backend (Python):**
    * Хранит профили пользователей и прогресс.
    * Выполняет тяжелые операции фотограмметрии (преобразование фото в 3D).
    * Раздает AssetBundles (дополнительные модели оборудования) по запросу.

---

## Технологический стек

### Client Side (Desktop)
* **Engine:** Unity 2022.3 LTS.
* **UI:** Unity UI Toolkit (USS/UXML).
* **Network:** UnityWebRequest (REST API взаимодействие).
* **Data:** JSON (сериализация локальных сцен).

### Server Side (Backend)
* **Language:** Python 3.10.
* **API:** FastAPI.
* **Database:** PostgreSQL (хранение метаданных).
* **Storage:** MinIO / Local Storage (хранение 3D-ассетов).

## Установка и запуск

### Для разработчиков (Source Code)

1.  **Клонирование репозитория:**
    ```bash
    git clone [https://github.com/username/AutoLab3D-Desktop.git](https://github.com/oron1ntedas/AutoLab3D.git)
    ```

2.  **Запуск Сервера (Backend):**
    Сервер необходим для авторизации в клиенте.
    ```bash
    cd src/backend
    pip install -r requirements.txt
    python main.py
    ```

3.  **Запуск Клиента (Unity):**
    * Откройте Unity Hub.
    * Нажмите "Add Project from disk" и выберите папку `src/client`.
    * Откройте сцену `Assets/Scenes/LoginScene.unity`.
    * Нажмите Play.

### Для пользователей (Build)
Актуальные сборки (`setup.exe`) находятся в папке `/builds` или в разделе [Releases].
